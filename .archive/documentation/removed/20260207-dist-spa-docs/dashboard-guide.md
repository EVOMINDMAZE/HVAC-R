# Understanding Your Dashboard

The ThermoNeural dashboard provides a quick overview of your recent activity and important metrics.

## Dashboard Sections

### Recent Calculations
Your most recent calculations are displayed here with the ability to:
- View calculation details
- Export to PDF
- Re-run with different values

### Quick Stats
- Total calculations this month
- Most-used calculator
- Saved refrigerant data

### Quick Actions
- Start a new calculation
- Access saved templates
- View your calculation history

## Customization

You can customize your dashboard by:
1. Clicking the settings icon in the top right
2. Selecting which widgets to display
3. Rearranging the order of sections

## Tips for Efficiency

- **Pin frequent calculators** to the top for quick access
- **Use keyboard shortcuts**: Press `?` to see available shortcuts
- **Enable notifications** to get alerts for completed calculations
