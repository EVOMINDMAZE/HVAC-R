# Refrigerant Properties Reference

Quick reference guide for common HVAC refrigerants and their key properties.

## Common Refrigerants

### R-410A (Puron)
- **Type**: HFC blend (R-32/R-125)
- **GWP**: 2088
- **ODP**: 0
- **Glide**: ~0.1°C (near azeotrope)
- **Apps**: Residential AC, heat pumps

### R-32
- **Type**: HFC, A2L (mildly flammable)
- **GWP**: 675
- **ODP**: 0
- **Glide**: 0 (pure compound)
- **Apps**: Mini-splits, VRF systems

### R-454B (Opteon XL41)
- **Type**: HFO/HFC blend, A2L
- **GWP**: 466
- **ODP**: 0
- **Glide**: ~1.5°C
- **Apps**: New R-410A replacements

### R-22 (HCFC-22) - *Phased Out*
- **Type**: HCFC
- **GWP**: 1810
- **ODP**: 0.055
- **Glide**: 0 (pure compound)
- **Apps**: Legacy AC systems (retrofit to R-407C or R-422D)

### R-404A
- **Type**: HFC blend
- **GWP**: 3922
- **ODP**: 0
- **Glide**: ~0.5°C
- **Apps**: Commercial refrigeration (transitioning to R-448A, R-449A)

## Key Terms

| Term | Definition |
|------|------------|
| **GWP** | Global Warming Potential (CO₂ = 1) |
| **ODP** | Ozone Depletion Potential (CFC-11 = 1) |
| **Glide** | Temperature difference between bubble and dew points in blends |
| **Azeotrope** | Blend that behaves like a pure compound (near-zero glide) |
